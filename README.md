# 🪔 Diwali Sales Analysis Using Python

## 📌 Project Overview

This project analyzes Diwali sales data using Python to understand customer purchasing behavior and identify business insights. The analysis helps companies improve customer experience, optimize marketing strategies, and increase sales during festive seasons.

## 🎯 Objectives

* Clean and preprocess the sales dataset.
* Perform Exploratory Data Analysis (EDA).
* Identify top-performing customer segments.
* Analyze sales based on gender, age group, occupation, state, and product category.
* Generate meaningful visualizations for business decision-making.

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📂 Dataset Information

The dataset contains customer purchase records during the Diwali festival season, including:

* User ID
* Gender
* Age Group
* State
* Occupation
* Product Category
* Marital Status
* Purchase Amount

## 📊 Analysis Performed

### 1. Data Cleaning

* Removed unnecessary columns.
* Handled missing values.
* Corrected data types.
* Removed duplicate records.

### 2. Exploratory Data Analysis (EDA)

#### Gender Analysis

* Compared sales between male and female customers.
* Identified the highest purchasing group.

#### Age Group Analysis

* Analyzed spending patterns across different age groups.
* Determined the most active customer segment.

#### State-wise Analysis

* Identified top states contributing to sales revenue.
* Compared order counts and total sales.

#### Occupation Analysis

* Examined purchasing behavior across professions.
* Found occupations with the highest spending.

#### Product Category Analysis

* Determined the most popular product categories.
* Analyzed category-wise revenue generation.

## 📈 Key Insights

* Female customers contributed more to overall sales.
* Customers aged 26–35 years showed the highest purchasing activity.
* Married customers were among the major buyers.
* States such as Uttar Pradesh, Maharashtra, and Karnataka generated significant sales.
* IT, Healthcare, and Aviation professionals were among the top-spending customers.
* Food, Clothing, and Electronics categories were highly popular.
